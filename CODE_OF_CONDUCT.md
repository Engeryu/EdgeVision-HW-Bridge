# Code of Conduct

## Our Standards

This project is a technical space. We expect all participants to engage with professionalism, intellectual honesty, and mutual respect.

**Expected behavior:**
- Communicate clearly and constructively, whether in issues, pull requests, or discussions.
- Provide actionable, specific feedback rather than vague criticism.
- Acknowledge when you are wrong or uncertain.
- Respect the time of maintainers and contributors.

**Unacceptable behavior:**
- Personal attacks, insults, or harassment of any kind.
- Dismissing contributions without explanation.
- Deliberately introducing breaking changes without discussion.
- Any form of discrimination based on background, identity, or experience level.

## Scope

This Code of Conduct applies to all project spaces: GitHub issues, pull requests, discussions, and any public communication referencing this project.

## Enforcement

Violations may result in comments being removed, contributions being rejected, or the contributor being blocked from the repository. Maintainers reserve the right to enforce these standards at their discretion.

To report an issue, open a private GitHub Security Advisory or contact the maintainer directly via GitHub.
